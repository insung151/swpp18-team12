## How to get started

- Clone repository

```
$ git clone https://github.com/swsnu/swpp18-team12.git
$ cd swpp18-team12
```

- Run backend server

```
$ cd backend
$ pip install -r requirements.txt
$ DJANGO_SETTINGS_MODULE=snuclub.settings.local python manage.py runserver
```
- Run backend server
```
$ npm install
$ npm start
$ npm test
```
